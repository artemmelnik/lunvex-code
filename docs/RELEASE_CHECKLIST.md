## Шаги для выполнения

1. **Обновить CHANGELOG.md:**
```bash
# Изменить CHANGELOG.md
## [0.1.0] - 2026-04-12
### Added
- Initial release of LunVex Code
- Core AI coding assistant functionality
- 13 Git tools integration
- Permission system with AUTO/ASK/DENY levels
- Project context support
- Comprehensive documentation
- 212 passing tests
- GitHub Actions CI/CD pipeline
```

2. **Создать тег:**
```bash
git tag -a v0.1.0 -m "Release v0.1.0: Initial release of LunVex Code"
git push origin v0.1.0
```

3. **Создать релиз на GitHub:**
   - Перейти в Releases → Draft a new release
   - Выбрать тег v0.1.0
   - Добавить описание релиза
   - Прикрепить файлы из dist/

4. **Настроить PyPI токен:**
   - Создать API токен на pypi.org
   - Добавить в GitHub Secrets как PYPI_API_TOKEN

5. **Публикация:**
```bash
# Тестовая публикация
twine upload --repository testpypi dist/*

# Продакшн публикация
twine upload dist/*
```
