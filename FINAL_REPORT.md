## 🔧 Технические изменения

### Конфигурация проекта (`pyproject.toml`)
```toml
# Было
keywords = ["ai", "coding", "assistant", "deepseek", ...]
lunvex-code = "deepseek_code.cli:main"
dsc = "deepseek_code.cli:main"
packages = ["deepseek_code"]

# Стало
keywords = ["ai", "coding", "assistant", "lunvex", ...]
lunvex-code = "lunvex_code.cli:main"
lvc = "lunvex_code.cli:main"  # Новый алиас
packages = ["lunvex_code"]
```

### Переменные окружения (`.env.example`)
```bash
# Новые переменные (рекомендуется)
LUNVEX_API_KEY=your_api_key_here
LUNVEX_MODEL=deepseek-chat
LUNVEX_BASE_URL=https://api.deepseek.com

# Старые переменные (обратная совместимость)
DEEPSEEK_API_KEY=your_api_key_here
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

### Основной клиент (`llm.py`)
```python
# Было
class DeepSeekClient:
    api_key = os.environ.get("DEEPSEEK_API_KEY")

# Стало
class LunVexClient:
    # Приоритет: LUNVEX_API_KEY, затем DEEPSEEK_API_KEY для обратной совместимости
    api_key = os.environ.get("LUNVEX_API_KEY") or os.environ.get("DEEPSEEK_API_KEY")
```
