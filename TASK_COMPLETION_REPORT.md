## 🚀 Инструкции по использованию

### 1. Быстрый старт:
```bash
# Запуск всех основных тестов
./run_async_tests.sh

# Просмотр списка всех тестов
python tests/test_all_async.py --mode=list

# Запуск всех тестов (включая требующие доработки)
python tests/test_all_async.py --mode=pytest
```

### 2. Разработка:
```bash
# Запуск конкретного тестового файла
python -m pytest tests/test_async_system.py -v

# Запуск конкретного теста
python -m pytest tests/test_async_system.py::test_async_read_file_tool -v

# Запуск с покрытием кода
python -m pytest tests/test_async_system.py --cov=lunvex_code.tools.async_file_tools
```

### 3. Интеграция в CI/CD:
```bash
# Пример для GitHub Actions
- name: Run async tests
  run: python tests/test_all_async.py --mode=pytest
```
