## Рекомендации для пользователей

### Для новых пользователей:
```bash
# Установка
pip install lunvex-code

# Настройка
export LUNVEX_API_KEY=your_key_here

# Использование
lunvex-code run
# или
lvc run
```

### Для существующих пользователей:
```bash
# Существующие настройки продолжат работать
export DEEPSEEK_API_KEY=your_key_here
lunvex-code run

# Рекомендуется перейти на новые переменные
export LUNVEX_API_KEY=your_key_here
```

### Для разработчиков:
```python
# Новый импорт
from lunvex_code.llm import LunVexClient
from lunvex_code.agent import Agent

# Старый импорт все еще работает (через симлинк)
```
